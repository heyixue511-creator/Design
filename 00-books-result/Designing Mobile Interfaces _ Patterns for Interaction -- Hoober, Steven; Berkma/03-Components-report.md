## 二、文献定位

该章讨论移动界面的组件，是构成移动交互系统的基础元素库。

## 三、核心观点

移动组件需要在可识别性、可触达性、状态反馈和平台一致性之间取得平衡。组件设计直接影响学习成本和操作可靠性。

## 四、核心概念

UI 组件；移动模式；可触达性；状态反馈；平台一致性。
