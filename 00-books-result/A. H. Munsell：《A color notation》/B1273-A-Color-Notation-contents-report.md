## 三、全书的知识递进

目录揭示Munsell并非只定义三个属性，而按以下步骤展开：

```text
颜色名称混乱
→ 分析视觉属性
→ 建立三维色立体
→ 创造数字记号
→ 记录多个颜色的关系
→ 分析平衡和构图
→ 进入教育、科学和工业应用
→ 将传统色名翻译为标准范围
```

这是从“单色识别”到“配色构成”，再到“社会应用”的递进结构。

## 四、第一章：从语言混乱到空间秩序

第一章的内部结构很明确：

1. 先证明传统色名不能精确沟通；
2. 再提出Hue、Value、Chroma；
3. 用球体和色彩树表达关系；
4. 最后以符号和数字形成记号。

因此，“色彩空间”不是先验图形，而是为解决语言和沟通问题建立的模型。

## 五、第二章：由单色到色组

`Color Score`位于记号之后，说明标准化的目标不止是给每个色样编号，还要记录：

- 多个颜色的组合；
- 配色序列；
- 冷暖、明暗和强弱分布；
- 作品或自然对象的综合色彩特征。

这使Munsell体系从分类工具转向设计分析工具。

## 六、第三章：由描述到构图判断

`Color Arrangements`把可测量属性用于视觉平衡、和谐和节奏。目录显示审美判断是全书第三阶段，而不是三属性定义本身。

研究时应避免把：

- 三属性测量；
- 色彩空间；
- 审美和谐法则

混为同一种科学可靠性。目录结构本身提示，后者属于在记号基础上的进一步应用和推论。
