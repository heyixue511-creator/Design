# 28-A TOUS LES FRANÇAIS 深度报告：自由法国宣言、抵抗记忆与原子时代

- 源文件：`00-books/Gallo, Max：The Poster in History, 1974/28-A TOUS LES FRANÇAIS.md`
- 文档性质：戴高乐宣言、解放海报与战争终结正文

## 内容核对
戴高乐1940年伦敦宣言号召法国人继续战斗；页面还讨论反犹宣传、犹太人大屠杀、Lidice等暴行、欧洲抵抗以及广岛和长崎原子弹。

## 深度分析
宣言用“失去一场战役而非战争”重新组织失败认知，并把分散法国人纳入未来胜利共同体。解放海报又将抵抗者牺牲纳入国家记忆，但可能弱化不同派别和普通民众。作者把原子弹主要写成历史分水岭，对日本平民经验和决策争议展开不足。

## 设计史意义
1. 分析流亡政治的文字传播；
2. 研究解放后抵抗记忆的国家化；
3. 将反犹海报视为灭绝政策语境；
4. 批判原子时代叙事的受害者缺席。

## 证据边界
宣言传播版本、抵抗贡献和原爆因果需专题史料；不得将纳粹宣传内容当作事实复述。

## 结论
本页从流亡号召到解放记忆和核武器，显示政治印刷如何重组失败、牺牲与战争终结的意义。
