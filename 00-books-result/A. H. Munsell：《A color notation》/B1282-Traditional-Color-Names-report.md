## 二、研究问题

本节追问：日常色名能否准确指向某一种颜色？

作者通过收集色样卡及其他来源中的两千多个命名颜色，发现色名并不都具有相同稳定性。

问题不只是名称多，而是：

- 同名可能覆盖多个颜色；
- 不同名称可能指向相近区域；
- 新商业名称缺少长期共同用法；
- 文化和行业语境会改变名称含义；
- 日常名称通常不提供精确明度和彩度。

## 三、两类颜色名称

文本将名称分为两类：

### 1. 传统色名

例如pink、brown、orange等，经过较长期使用，通常与相对明确的颜色区域或范围相关。

### 2. 新近色名

使用历史较短、尚未形成稳定公共理解，不能可靠归为传统色名。

这一分类强调词汇稳定性来自社会使用历史，而非名称自身天然准确。

## 四、传统色名不是单一坐标

即使是相对稳定的传统色名，也只对应色彩空间中的一个区域，而不是唯一点位。

例如两个都被称为pink的颜色，可能在：

- 色相倾向；
- 明度；
- 彩度

上明显不同。

因此，传统名称适合粗略分类，却不适合精密制造、复制和学术记录。

## 五、为何建议学生避免传统色名

作者建议色彩学习者在专业工作中尽量避免只使用传统名称，因为它们不能准确说明两个相似颜色之间的差别。

更精确的方法是使用三维属性：

```text
Hue + Value + Chroma
```

这使颜色描述从类别词转向可比较位置。

## 六、日常语言仍有价值

作者并未要求完全废除传统色名。面对没有学习Munsell记号的人时，传统名称仍有沟通作用。

因此，体系建立的是两层语言：

- 技术层：用标准记号精确定位；
- 公共层：用熟悉词汇快速沟通。

关键任务是建立两者之间的对应关系。

## 七、二十四个常用色名组

文本说明，后续页面总结二十四个常用名称。每一名称代表一个颜色群组，并给出大致边界。

这些组包括粉红、栗色、橙、棕、象牙、柠檬黄、橄榄绿、祖母绿、天蓝、海军蓝、紫罗兰、灰、白等类别。

完整组名和数值范围主要保存在B1284表格中，当前入口文件没有全部列出。

## 八、研究从名称转向范围

这一方法的重要变化是：不再争论某个词“真正”是哪一种颜色，而是调查社会使用后，为它划定近似范围。

其逻辑是：

1. 收集带名称的色样；
2. 观察同名样本分布；
3. 在Munsell空间中定位；
4. 给出色相、明度和彩度范围；
5. 将相近名称编入同一群组或建立交叉引用。

当前片段没有详细说明抽样和统计方法，后续表也不能替代方法报告。

## 九、商业色名与标准化

颜色名称常用于：

- 服装和纺织；
- 油漆；
- 化妆品；
- 家居；
- 印刷；
- 商品目录；
- 广告和品牌传播。

商业需要富有吸引力的名称，但生产需要可复核的坐标。Munsell对照系统试图处理营销语言与制造语言之间的冲突。

## 十、文化意义与技术精度

旧报告说日常色名“富含文化意义”，总体方向合理，但当前源文主要分析使用稳定性和范围，并没有系统研究色名的象征、情感或跨文化意义。

因此不能仅凭本节推导：

- 某色名在不同文化中的象征；
- 色名的性别或阶级含义；
- 全球语言之间的等值关系。

这些需要语言学、人类学和跨文化研究补充。

## 十一、“传统”的地域边界

文本没有明确说明两千多个色名样本的完整地域、行业和年代构成。根据书籍语境，它们很可能主要来自英语和美国商业色样体系。

因此，“传统色名”不能理解为全人类通用传统，而应理解为特定资料集合中的历史用法。

## 十二、标准化与语言损失

将色名映射为坐标能提高精度，但也可能减少：

- 地方记忆；
- 材料来源；
- 物象联想；
- 情感语气；
- 历史和商业故事。

设计实践往往需要同时保留：

- 标准编号用于生产；
- 文化名称用于叙事和传播。

## 十四、关键概念

| 概念 | 本节含义 | 研究用途 |
|---|---|---|
| Traditional Color Name | 长期使用、对应相对稳定范围的名称 | 术语史、商业史 |
| Recent Color Name | 尚未形成稳定社会边界的新名称 | 品牌命名、市场传播 |
| Color Range | 一个名称覆盖的色彩空间区域 | 分类、数据映射 |

| Color Group | 将近似名称和范围组织为群组 | 检索、术语管理 |

## 十五、可提炼的核心观点

1. 传统色名通常对应颜色范围，而非单一颜色。
2. 新近商业名称因使用历史不足而更不稳定。
3. 精确区分相似颜色需要Hue、Value、Chroma三维记号。
4. 日常名称仍是面向非专业公众的重要沟通工具。
5. 标准体系的任务之一是建立技术坐标与公共词汇之间的翻译。
6. 二十四个色名组是经验分类，不是全球普遍语言。
7. 当前文件在Pink组说明处截断，必须与B1283合读。

## 十九、结论

《Traditional Color Names》并非简单反对日常语言，而是建立一种双层沟通机制：用Munsell记号保证专业精度，用传统色名维持公共可理解性。

其设计史意义在于揭示颜色标准化不仅是测量问题，也是词汇、市场、文化和信息翻译问题。
