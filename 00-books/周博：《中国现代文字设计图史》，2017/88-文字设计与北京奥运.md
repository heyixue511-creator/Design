# 文字设计与北京奥运 2008年的北京奥运会是中国改革开放和现代化进程中的一件大事，同时也是中国当代设计发展和国家形象塑造的一个重要节点。全世界都在通过奥运会的视觉设计重新理解中国文化，而全世界的设计同行也都通过奥运会对中国当代设计水准有了一个基本的认识。从北京奥运会的视觉设计中，我们可以看到文字设计对于塑造中国形象、传播中国文化的重要价值。 北京奥运会的会徽“中国印·舞动的北京”，受到了中国传统的篆刻艺术的影响。会徽的核心图形是一个阴文的运动员形象，似乎是在向前奔跑、迎接胜利。这个形象又让人联想起“文”字和“京”字，这样，中国悠久的文化和北京奥运会的主题就都具备了。而北京残奥会的会徽“天地人”则以天、地、人和谐统一为设计理念，把中国的文字、书法与残疾人的奥林匹克运动精神融为一体。会徽的主体形象很像个“之”字，我们在“书圣”王羲之书写的众多“之”字中可以找到与会徽相近的字形，但我们在临写王羲之的书法时，其实是很难想到它们在设计上还会有这些妙用的。北京奥运会的体育图标，也是很能够体现中国古老的文化特征，同时又是在设计和视觉传播效果上非常成功的一组作品。这些图标中的主体当然都是“人”的运动形象，但是那些古朴苍浑又充满生命力的线条却来自《散氏盘》和《毛公鼎》中的大篆。篆刻、篆书属于书法中“碑学”的范畴，王羲之的行书则属于书法中“帖学”的范畴。我们常说，能够代表中国最核心的东西是汉字，而中国最伟大的汉字艺术传统都隐藏在书法中，北京奥运会的会徽设计再次印证了这一点。2022年北京冬奥会会徽“冬梦”和冬残奥会会徽“飞跃”的设计，虽然把“冬”“飞”二字抽象化，但依然在延续关于 # Beijing 2008 TM© 1.2008年北京奥运会会徽，主要设计者为张武。图片来源：2008年奥运会官方网站。 2.2008年北京残奥会会徽，刘波设计。 图片来源：刘波提供。 ![](images/0b90c4f2c95ecc94e7e3160f47268b40bfef6eae7708c096d4fd9e804cd27cdb.jpg) ![](images/4a771a2385027bf7be3da3c11b64093ed6289c89927c60c06c417b83ba25dd3b.jpg) ## Paralympic Games ![](images/6c53a529aba1abac65b49b476f38d7ef0bf657ad92b001c726bdfbb601b1d61b.jpg) # HSRTFSMR ![](images/4f63afef64c35c6bcd4eb953f66b10f5ec84474647eae17464e2b5f2162c4168.jpg) 2008.8.8—8.24www.beijing2008.cn
