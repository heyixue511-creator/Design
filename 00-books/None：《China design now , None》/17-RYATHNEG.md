# RYATHNEG

This is not a comprehensive bibliography on design and architecture in contemporary China, but is intended as a guide to some of the sources in and beyond this book. This bibliography includes books, papers/articles and journals, as well as entries for magazines and websites. It is arranged by discipline, although some materials have relevance to more than one section.
