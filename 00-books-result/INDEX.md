# `00-books-result/` 深度报告索引

本索引记录 `00-books/` 中书籍和章节文档的深度报告产出。

## 目录结构原则

`00-books-result/` 与 `00-books/` 保持相同的相对目录结构：

```text
00-books/某书名/某章节.md
00-books-result/某书名/某章节-report.md
```

说明：个别原镜像文件名被系统拦截时，已使用备用安全文件名保存，并保持报告编号连续。目前备用文件名包括：B0167、B0251、B0278。
