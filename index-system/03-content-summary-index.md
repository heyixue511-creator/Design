# 03 内容摘要索引

本索引用于说明每本文献“到底讲什么、研究什么、对教材有什么用”。它是从文献清单走向教材写作的关键环节。

## 一、字段模板

| 编号 | 文献编号 | 文献题名 | 核心内容 | 研究对象 | 主要观点 | 论证路径 | 重要章节 | 关键词 | 适用章节 | 适用程度 | 使用价值 |
|---|---|---|---|---|---|---|---|---|---|---|---|
| S001 | B001 |  |  | 人物 / 作品 / 制度 / 观念 / 技术 / 社会背景 |  |  |  |  |  | 核心文献 | 可作为基础叙述框架 |
| S002 | P001 |  |  |  |  |  |  |  |  | 重要文献 | 可用于深化理论分析 |

## 二、适用程度

| 等级 | 含义 | 使用方式 |
|---|---|---|
| 核心文献 | 可直接构成章节框架或核心论证 | 必读、重点引用 |
| 重要文献 | 对某一主题有关键支撑 | 局部精读、重点转述 |
| 辅助文献 | 提供案例、背景、材料 | 选择性使用 |
| 背景文献 | 帮助理解历史语境或学术脉络 | 备查、补充阅读 |
| 待核文献 | 信息不完整或需核对 | 暂不进入核心论证 |

## 三、摘要撰写规范

建议每条摘要控制在 100—300 字，包含三点：

1. 文献讨论的核心问题；
2. 作者提出的主要判断；
3. 对《现当代设计史》教材写作的用途。

## 四、示例格式

| 编号 | 文献 | 核心内容 | 适用章节 | 使用价值 |
|---|---|---|---|---|
| S001 | 某设计史专著 | 梳理现代设计从工业革命到包豪斯的发展，并将设计置于工业生产、社会改革与视觉文化的关系中讨论。 | 现代设计史章节 | 可作为基础叙述框架 |
| S002 | 某理论论文 | 讨论设计与社会生产、技术系统、公共生活之间的关系，强调设计不只是形式问题。 | 设计理论章节 | 可用于深化理论分析 |

## 五、解决的问题

这一层解决：这本文献到底有什么用，是否值得进入教材正文、脚注、推荐阅读或教师备课资料。
