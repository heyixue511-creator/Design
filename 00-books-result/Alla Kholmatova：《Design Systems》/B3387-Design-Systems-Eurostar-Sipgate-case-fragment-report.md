## 二、原文实际包含什么

Eurostar 部分说明 Dan Jackson 在 2016 年 8-9 月和 2017 年 3 月分享了团队建立第一个 pattern library 的情况：初期面临优先级和鼓励团队贡献的挑战，后来获得资源建立专门团队。Sipgate 部分说明 Tobias Ritterbach 和 Mathias Wegener 分享经验：Sipgate pattern library 建于 2015 年，但一年后因产品团队沟通不足导致模式过多，后来开始建立新的 pattern library，以统一多个产品网站的设计语言。
