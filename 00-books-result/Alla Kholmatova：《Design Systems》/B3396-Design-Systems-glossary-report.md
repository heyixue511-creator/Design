## 二、原文实际包含什么

文件说明保持 glossary 是共享和演化设计语言的有效实践。建立 glossary 会迫使团队意识到并明确表达所使用的词语。Intercom 案例中，团队用 glossary 确保“from code to customer”使用相同语言。
