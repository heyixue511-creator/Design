## 二、文献定位

该文是对 Kirkham 与 Weber 主编的设计史教材型大书的评论，重点关注其亚洲尤其日本设计史章节如何回应全球、跨国和多中心设计史诉求。它非常适合用于《现当代设计史》教材写作时反思“世界设计史”或“全球设计史”的组织方式。

## 三、核心内容摘要

Čapková 肯定该书作为大型设计史教材的组织能力、图像质量和包容性目标，但也指出其“全球化”路径主要采用从总体大叙事到各地区案例的方式。她借 Anna Calvera 关于全球化设计史的两种路径，区分了“大型共同叙事”和“从地方差异出发的比较叙事”。她认为该书虽然努力纳入多地区历史，但仍可能因印刷书的固定结构、地区均衡逻辑和材料 / 媒介分类方式而弱化跨国流动、混杂身份、移民网络和跨区域设计者的作用。尤其在日本现代主义问题上，跨国训练、海外工作、外来设计者在日本的实践等因素应被更充分呈现。

## 四、研究对象

- 设计史教材写作；
- 全球设计史与跨国设计史；
- 日本现代设计史；
- 地区结构、媒介结构和主题结构的教材组织问题；
- 横向 / 多声部艺术史方法。

## 六、核心概念

| 概念 | 英文 | 教材解释方向 |
|---|---|---|
| 全球设计史 | global design history | 关注设计在全球范围内的联系、流动、比较和结构不平等。 |
| 跨国设计史 | transnational design history | 研究设计者、物、技术、图像和观念跨越国家边界的流动。 |
| 横向艺术史 | horizontal art history | 反思中心—边缘结构，强调多中心、多声部和地方生产的艺术史方法。 |
| 多中心叙事 | multicentred narratives | 不以单一欧美中心为主轴，而呈现多地域互动的历史叙述。 |
| 教材结构 | textbook structure | 教材如何通过章节、地区、媒介、主题组织知识的方式。 |

## 七、论证路径

文章先说明 Kirkham 与 Weber 书籍的全球设计史目标，再用 Calvera 的两种全球化设计史路径进行定位。随后，作者从结构和亚洲 / 日本章节出发，评价该书的优点与局限。最后，她引入 Elkins、Piotrowski 等关于全球艺术史和横向艺术史的讨论，指出该书应更清晰地说明其全球、跨国和世界史方法。

## 八、可用于《现当代设计史》的位置

| 教材位置 | 使用方式 |
|---|---|
| 教材导论 | 用于说明教材不是资料堆砌，而是历史结构设计。 |
| 全球设计史章节 | 用于比较大叙事与地方比较两种写法。 |
| 日本现代设计史章节 | 用于强调跨国网络与设计者移动。 |
| 方法论章节 | 用于讨论 printed textbook 与 digital format 对设计史结构的影响。 |

## 九、教学转化建议

- 课堂问题：全球设计史应该写成一部大叙事，还是由地方差异组成的多声部结构？
- 讨论题：教材章节结构会不会改变我们对历史的理解？
- 作业建议：选择一个国家或地区设计史案例，补充其跨国流动和网络关系。
- 教学难度：中级—高级。
- 适合对象：本科高年级、研究生。

## 十、待核问题
