## 二、文献定位

该文介绍 Dunne 与 Raby 的 Placebo Project，是批判设计和设计研究中极具代表性的案例。它把设计研究从实验室带入日常家庭生活，通过开放性原型探索人们对电磁场、电子物和不可见技术环境的想象、焦虑和叙事。

## 三、核心内容摘要

Placebo Project 制作了八个原型对象，用于研究人们如何理解家中电子物产生的不可见电磁场。这些物件既像家具或日常物，又具有某种特殊功能，例如感应电磁场、保护磁性纪念物、排出身体电力或靠泄露辐射发光。项目并不试图科学地证明或解决电磁波问题，而是像安慰剂一样改变人们对不可见技术环境的感知。Dunne 与 Raby 通过把物件放入志愿者家中并进行访谈，收集人们围绕电子物“秘密生活”形成的故事和态度。

## 四、研究对象

- 电磁场与家庭电子物；
- 批判设计原型；
- 设计研究进入日常生活；
- 用户叙事和技术想象；
- 安慰剂式设计对象。

## 七、论证路径

文章先说明 Placebo Project 的实验目标和原型设置，再介绍招募志愿者、收养物件、访谈和摄影记录过程。随后以 Nipple Chair 等案例展示人们如何赋予物件生命感、情感关系和环境感知意义。文本最终说明设计可以在科学解决之外，创造新的感知、叙事和技术关系。

## 九、教学转化建议

- 课堂问题：设计是否一定要解决实际问题？
- 讨论题：安慰剂式设计对象是否也能产生真实经验？
- 作业建议：设计一个能让人意识到不可见环境因素的开放性原型。
- 教学难度：中级—高级。
- 适合对象：本科高年级、研究生。
