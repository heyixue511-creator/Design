# 深度报告与中央索引衔接规则

本文件规定 `00-books-result/` 深度报告如何进入《现当代设计史》综合索引系统。它不替代深度报告生成规范，也不替代各索引表的字段定义，而是规定两者之间的数据流、编号权属、去重和复核边界。

```text
00-books/ 源 Markdown
        ↓ 逐文件真实读取
00-books-result/ 深度报告（精读层）
        ↓ 候选项抽取、证据核验、实体归并
index-system/ 01-12（中央索引层）
        ↓ 教材章节组织、教学转化、引用管理
《现当代设计史》写作与教学
```

## 一、职责边界

| 层级 | 负责内容 | 不负责内容 |
|---|---|---|
| 源 Markdown | 保存原始文本、OCR 结果、图注和文献结构 | 不直接承担教材归类 |
| 深度报告 | 逐文件核验、语义分析、知识要素拆解、证据边界和索引候选项 | 不独立分配跨文献全局实体编号 |
| 中央索引 | 跨文献归并、统一编号、主题与章节映射、版本和引文管理 | 不用索引摘要替代原文核验 |

## 二、编号权属

- `B`、`P`：文献编号，应与现有源文和报告编号保持一致。
- `TH`：主题分类编号，由 `02-theme-index.md` 维护。
- `V`、`C`、`N`、`K`：观点、概念、人物和案例的全局编号，由对应中央索引维护。
- `Q`、`Fig`、`T`：引文、图像和时间事件的全局编号，由 `10`、`11` 号索引维护。
- 报告内的表格行号或 A/B/C 分组只表示局部结构，不具有全局编号效力。

## 三、报告入库字段

每份可靠报告进入索引抽取队列时，至少提供以下信息：

| 字段 | 要求 |
|---|---|
| 文献编号 | 与源文及有效报告一一对应 |
| 源文件路径 | 使用仓库内稳定相对路径 |
| 报告路径 | 指向唯一有效报告 |
| 文献类型 | 章节、前言、目录、图版、论文、访谈等 |
| 完整性 | 完整、截断、OCR 缺损、空文本、待合读等 |
| 主题候选 | 对应既有 `TH` 主题，必要时提出待审新主题 |
| 观点候选 | 主张、证据、限定条件和作者立场 |
| 概念候选 | 中文主名、原文名、别名、定义语境 |
| 人物候选 | 姓名、原文名、身份、在本文中的作用 |
| 案例候选 | 名称、类型、时间、地点、责任者和论证用途 |
| 引文候选 | 原文、定位信息、版本状态和可核性 |
| 图像候选 | 图号、图注、来源、版权与是否读取图像本体 |
| 教材用途 | 章、节、知识单元、教学功能和使用限制 |
| 证据等级 | 可直接引用、需版本核验、仅作线索或不可确认 |

## 五、实体归并与去重

1. 先按源文件路径确认文献身份，再按文献编号核对，不以报告文件名猜测映射。
2. 同一源文只能有一份有效报告；旧版、误映射和重复报告不得同时参与索引抽取。
3. 人物、机构、作品和概念按“规范中文名 + 原文名 + 别名”归并，不能只按英文字符串匹配。
4. 同名异人、同名作品和不同版本概念必须保留时间、地域、责任者或语境限定。
5. 多份报告支持同一实体时，中央索引保留一个主条目，并记录全部来源文献编号。
6. 报告之间存在事实冲突时，不以出现次数决定真伪；应记录争议并回到源文、版本和责任者信息复核。

## 六、教材转化要求

索引条目必须能回答以下问题：

1. 它对应教材哪一章、哪一节或哪个知识单元？
2. 它承担概念解释、历史叙事、案例论证、争议讨论、方法训练还是延伸阅读？
3. 它的证据来自哪份源文和哪份有效报告？
4. 它是否可以直接引用，是否存在页码、译本、OCR 或版权限制？

无法回答上述问题的条目可以进入待核队列，但不得直接进入教材正文引用队列。

## 七、执行顺序

1. 审核源文与有效报告的一一对应关系。
2. 排除停止使用、误映射、重复和无对应源文的报告。
3. 从可靠报告抽取主题、观点、概念、人物、案例、引文、图像和时间事件候选项。
4. 进行中英文实体规范化与跨报告去重。
5. 分配或复用中央索引编号。
6. 写入 `01-12` 对应索引，并建立来源反向链接。
7. 复核教材章节、教学功能、证据等级和引用边界。
8. 记录本轮新增、合并、冲突和待核项目，保证后续可追踪。

## 八、禁止事项

- 不得从文件名、目录名或旧报告推断源文内容。
- 不得把报告摘要当作原文引文。
- 不得为无页码 Markdown 虚构纸本页码。
- 不得只抽取英文实体而遗漏中文书籍中的中文人物、概念和案例。
- 不得让同一实体在不同索引中拥有互相冲突的主名称或编号。
- 不得把“待核”“可能”“疑似”信息改写成确定事实。
- 不得让已停止使用的报告继续进入中央索引。
