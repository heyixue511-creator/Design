# `00-Paper-result/` 论文深度报告目录

本目录用于存放 `00-Paper/` 中 72 个 Markdown 文档的逐篇深度报告。

## 一、报告目标

每一份报告不只是摘要，而是将原始论文材料转化为《现当代设计史》教材写作可直接使用的资料卡、观点卡、概念卡、章节适配卡和教学卡。

## 二、报告命名规则

```text
P001-文献短名-report.md
P002-文献短名-report.md
...
P072-文献短名-report.md
```

编号 `P001—P072` 与 `paper-paths.txt` 的文献顺序对应。

## 三、单篇报告结构

每篇报告建议包括：

1. 基础书目信息；
2. 原始文件路径；
3. 文献类型与重要性等级；
4. 核心内容摘要；
5. 研究对象；
6. 主要观点；
7. 论证路径；
8. 核心概念；
9. 可提炼观点索引；
10. 可用于教材章节；
11. 可用案例；
12. 教学转化建议；
13. 可引用页码或段落线索；
14. 待核问题。

## 四、当前进度

进度见 [`PROGRESS.md`](PROGRESS.md)。

## 五、说明

本目录的报告将优先处理 A 类核心理论文献，再逐步覆盖全部 72 篇论文。
