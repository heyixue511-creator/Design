## 二、文献定位

该章将“设计语言”作为理解产品、服务、组织和软件界面设计的重要概念。它说明设计语言不是单纯风格，而是由元素、组织原则和情境变化组成的意义生成系统，能够帮助用户解释、学习和融入设计物。

## 三、核心内容摘要

作者把设计语言类比为自然语言：自然语言让人生成和理解句子，设计语言让设计师创造物，也让用户解释并使用物。设计语言可以在建筑、产品、城市、服务、组织和软件中出现，很多时候无意识地演化，但当它被有意识地理解、发展和应用时，可以显著改善人、环境和物之间的交互。

章节以 Nantucket 房屋、Xerox 复印机、Macintosh 界面、儿童电子产品和个人通信器为例说明，设计语言由形状、颜色、材质、动作、隐喻等元素，组合原则，以及随情境调整的规则组成。好的设计语言不仅让对象表达“它是什么、能做什么、如何使用”，还让对象进入人的日常经验。作者特别强调 black-box 与 transparent-box 的区别：设计语言可以让功能被适度揭示，使用户通过使用学习和发展能力。

## 五、核心概念

| 概念 | 英文 | 教材解释方向 |
|---|---|---|
| 设计语言 | design language | 由视觉、形式、功能和交互元素构成的意义生成系统。 |
| 设计元素 | design elements | 形状、颜色、质感、动作、隐喻等构成设计语言的基本单位。 |
| 组织原则 | organizing principles | 规定元素如何组合并生成意义的规则。 |
| 情境限定 | qualifying situations | 根据使用情境调整设计语言的条件。 |
| 黑箱设计 | black-box design | 隐藏功能机制、只提供封闭操作的设计。 |
| 透明箱设计 | transparent-box design | 适度揭示功能机制，使用户能学习和掌握系统的设计。 |

## 六、教材使用建议

可用于“从风格到设计语言”章节，说明现代设计中的统一性不只是视觉风格一致，而是通过符号、行为、功能、学习和体验建立系统性意义。也适合用于讲解 GUI、产品家族、企业识别和服务体验的一致性。

## 七、教学转化

课堂问题：Apple、Xerox 复印机或某一品牌产品线中的“设计语言”具体由哪些元素构成？作业可要求学生拆解一个产品家族的设计语言，并提出一个新产品如何延续或更新这种语言。
