# 01 总文献索引表

本索引用于保证文献信息准确、可引用、可查找，是所有后续主题索引、观点索引、章节索引的基础。

## 一、字段模板

| 编号 | 作者 | 译者 | 题名 | 原书名 / 原文题名 | 出版地 | 出版社 / 期刊 | 出版年份 | 所用版本年份 | 文献类型 | ISBN / DOI | 页码范围 | 文件路径 | 引用格式 GB/T 7714—2015 | 重要性等级 | 备注 |
|---|---|---|---|---|---|---|---:|---:|---|---|---|---|---|---|---|
| B001 |  |  |  |  |  |  |  |  | 专著 |  |  | `00-books/...` |  | A |  |
| P001 |  |  |  |  |  |  |  |  | 论文 |  |  | `00-Paper/...` |  | A |  |

## 二、文献类型建议

| 类型 | 说明 |
|---|---|
| 专著 | 单一作者或多作者完整著作 |
| 译著 | 有中文译本的外文著作 |
| 论文 | 期刊论文、会议论文、专题文章 |
| 论文集 | 多篇论文合编成书 |
| 教材 | 可直接用于教学框架的教材型文献 |
| 报告 | 机构报告、研究报告、政策报告 |
| 年鉴 | 年度资料或统计性文献 |
| 档案 | 历史档案、原始材料 |
| 访谈 | 人物访谈、口述史材料 |
| 书评 | 对重要书籍的评论性材料 |

## 三、编号规则

| 来源 | 编号前缀 | 示例 |
|---|---|---|
| `00-books/` | B | B001 |
| `00-Paper/` | P | P001 |
| 待核资料 | D | D001 |

## 四、填写要求

1. 作者、题名、出版年份必须优先核对。
2. 外文文献保留原文题名；有中文译名时可在备注中记录。
3. 翻译类文献必须记录译者与所用译本年份。
4. 引用格式统一使用 GB/T 7714—2015，后续可根据学校规范调整。
5. 文件路径必须能指向仓库内具体 Markdown 文件。

## 五、解决的问题

这一层解决：这本文献是谁写的、什么时候出版、在哪里能找到、如何引用、能否作为教材写作依据。
