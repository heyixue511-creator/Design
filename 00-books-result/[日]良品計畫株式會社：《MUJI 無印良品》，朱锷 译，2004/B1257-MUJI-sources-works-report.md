## 二、文件性质

《出处·作品一览》不是关于无印良品理念的论述，而是全书图像和案例的元数据索引。它把正文中分散出现的对象重新对应到：

- 页码；
- 项目或产品名称；
- 年份；
- 尺寸与型号；
- 创意总监；
- 艺术总监；
- 产品设计师；
- 平面设计师；
- 摄影师；
- 文案和编辑人员；
- 店铺、展览及出版机构。

它最重要的作用是纠正“无印良品由单一设计大师创造”的误读，展示大量产品和传播项目背后的多人协作。

## 四、设计责任的分层

文件清楚区分多个角色：

| 角色 | 主要责任 |
|---|---|
| 创意总监 | 确立项目整体方向和品牌原则 |
| 艺术总监 | 组织视觉表现和设计执行 |
| 产品设计师 | 负责产品形态、结构与使用 |
| 平面设计师 | 完成版式、文字与视觉系统 |
| 摄影师 | 建构图像和项目记录 |
| 文案 | 形成传播语言 |
| 编辑／制作 | 组织出版和项目落地 |
| 室内设计师 | 负责店铺和空间项目 |

这一分层对于设计史非常重要，因为同一个项目可能同时涉及策划、图像、文字、产品和生产，不能只署名最知名的人物。

## 五、案例年代与品牌阶段

索引跨越1980年代至2000年代，可用于建立品牌发展年表：

1. 1980—1984：早期广告、命名与材料原则；
2. 1990年代：国际门店、跨文化主题海报和产品扩张；
3. 2000年代：原研哉艺术指导、产品系统、住宅、展览及全球城市传播。

但索引并非完整企业年表；它只记录本书采用的图版与案例，未入选的项目不能因此被认为不存在。

## 六、OCR和转写问题

源文件存在多种质量风险：

- 人名同音异写，如“麹谷／麴谷／鞠谷”等；
- “写真”等字段可能因断句与前后条目错位；
- 产品名称、尺寸、年份和页码挤在同一行；
- 日文、中文和英文混排时字符丢失；
- 图片和条目边界不清；
- 工具返回内容因体量过大而截断。

因此，正式引用必须逐项回到原图或出版物校核，不能批量复制当前OCR作为最终著录。
