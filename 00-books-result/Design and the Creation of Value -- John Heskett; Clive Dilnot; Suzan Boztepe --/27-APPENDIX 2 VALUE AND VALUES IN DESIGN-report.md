## 二、文献定位

该附录区分 value 与 values，适合用于说明设计价值既包括经济效益，也包括伦理、社会、文化和生活价值。

## 三、核心观点

设计中的 value 指向效用、经济和实践成果；values 指向规范、信念和价值观。设计价值研究必须同时处理二者，否则会把设计简化为市场收益或美学偏好。

value；values；伦理价值；经济价值；设计评价。
