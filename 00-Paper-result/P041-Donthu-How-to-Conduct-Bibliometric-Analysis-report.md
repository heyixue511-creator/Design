## 二、文献定位

该文是文献计量分析方法的高实用性指南，适合用于用户此前提出的“当代艺术史研究中的文献计量方法：知识结构、应用场景与趋势反思”方向，也可用于《现当代设计史》资料库的研究方法章节。

## 三、核心内容摘要

文章指出，文献计量分析是一种用于探索和分析大量科学数据的严格方法，能够揭示某一领域的演化、知识结构和新兴主题。作者解释了文献计量分析兴起的原因：科学数据库如 Scopus、Web of Science 的可用性，以及 Gephi、Leximancer、VOSviewer 等软件的发展。文章区分文献计量分析、系统综述和元分析，并指出文献计量适用于研究范围较宽、数据量较大、人工阅读难以处理的领域。其技术工具箱包括 performance analysis 和 science mapping，后者又涉及引文分析、共引分析、文献耦合、共词分析和网络分析。

## 四、研究对象

- 文献计量方法；
- 绩效分析和科学知识图谱；
- 引文、共引、文献耦合、共词和网络分析；
- Scopus、Web of Science、VOSviewer、Gephi 等工具；
- 系统综述、元分析和文献计量的比较。

## 六、核心概念

| 概念 | 英文 | 教材解释方向 |
|---|---|---|
| 文献计量分析 | bibliometric analysis | 使用定量方法分析文献、引用、作者、关键词和知识网络。 |
| 绩效分析 | performance analysis | 统计研究主体的产出、影响和贡献。 |
| 科学图谱 | science mapping | 分析知识结构、主题关系和研究前沿的网络方法。 |
| 共引分析 | co-citation analysis | 根据文献被共同引用的关系识别知识基础。 |
| 文献耦合 | bibliographic coupling | 根据文献共同引用相同参考文献的关系识别研究前沿。 |
| 共词分析 | co-word analysis | 根据关键词共同出现关系识别主题结构。 |

## 七、论证路径

文章先说明文献计量方法在商业研究中的兴起和应用价值，再比较文献计量、系统综述和元分析的适用条件。随后介绍文献计量技术工具箱，包括绩效分析与科学图谱。文章进一步讨论指标、聚类算法、网络分析和操作流程，为研究者提供逐步指南。

## 九、教学转化建议

- 课堂问题：什么时候适合使用文献计量分析？
- 讨论题：文献计量图谱能否替代人工阅读？
- 作业建议：选取一个设计史主题，从 Web of Science 或 Scopus 导出文献，制作共词或共引图谱。
- 教学难度：中级—高级。
- 适合对象：本科高年级、研究生。
