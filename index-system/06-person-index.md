# 06 人物索引

本索引用于整理《现当代设计史》中必须讲述或可作为拓展阅读的人物，包括学者、设计师、艺术家、教育家、建筑师、策展人和思想家等。

## 二、人物类型

| 类型 | 说明 | 可能章节 |
|---|---|---|
| 设计史学者 | 建构设计史、设计研究或设计理论的人物 | 学科导论、方法论 |
| 设计师 | 有代表性设计作品或实践的人物 | 现代设计、平面设计、产品设计 |
| 艺术家 | 与现代设计、视觉文化或先锋派有关的人物 | 现代主义、先锋派 |
| 教育家 | 推动设计教育制度和教学改革的人物 | 包豪斯、乌尔姆、设计教育 |
| 建筑师 | 与现代建筑、空间设计、设计思想相关的人物 | 现代主义、城市与空间 |
| 思想家 | 对设计理论、技术、社会、媒介有影响的人物 | 理论基础、方法论 |

## 四、填写要求

1. 人物索引不只记录生平，更要记录“教材中怎么讲”。
2. 对同一人物，应区分历史贡献、理论贡献和教学重点。
3. 人物必须关联到文献、概念、案例或章节，否则不进入核心人物表。
4. 容易误解的人物需在“注意事项”中说明。

## 五、解决的问题

这一层解决：教材中哪些人物必须讲，如何讲，放在哪一章讲，讲授重点是什么。
