| 引用目的 | 说明 |

## 四、译本管理要求

1. 外文经典著作必须记录原版年份与所用版本年份。
2. 有中文译本时，记录译者、出版社、出版年份。
3. 同一概念在不同译本中的译名差异需要记录，例如 modernity、modernism、material culture 等。
4. 正文页码必须对应实际使用版本，不能混用不同版本页码。
5. 教材中直接引用外文原文时，应同时保存原文和译文。

## 五、可引用材料记录规范

建议每条引文都包括：

```md
### Q001 引文标题

- 文献编号：
- 原文：
- 中文译文：
- 页码 / 段落：
- 可用章节：
- 引用目的：
- 是否需要核对原书：是 / 否
```

## 六、解决的问题

这一层解决：避免版本、译名、页码、引用方式混乱，使教材正文、注释和参考文献保持一致。
