# design-history

This repository contains notes, excerpts, and study materials related to design history.

本仓库用于支持《现当代设计史》相关课程、论文、资料卡和参考文献整理，主要包含设计史、设计研究、设计理论、设计文化、HCI、全球设计史、可持续设计与转型设计等资料。
