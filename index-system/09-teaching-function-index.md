# 09 教学功能索引

本索引用于将学术文献转化为教材内容、课堂活动、阅读任务和作业设计。

## 二、教学难度

| 难度 | 说明 | 适合对象 |
|---|---|---|
| 入门 | 概念清晰、叙述性强、适合作为基础阅读 | 本科低年级 |
| 中级 | 有一定理论性，需要教师导读 | 本科高年级 |
| 高级 | 理论密度高、方法复杂、适合研究型学习 | 研究生 |

## 三、教学用途

| 用途 | 说明 |
|---|---|
| 概念解释 | 用于讲清楚一个核心术语或基本问题 |
| 课堂讨论 | 适合提出争议性问题，引导学生讨论 |
| 案例分析 | 通过案例解释历史现象或理论概念 |
| 拓展阅读 | 为学有余力的学生提供延伸文献 |
| 研究训练 | 训练文献阅读、资料卡、论文写作能力 |
| 参考文献 | 作为教师备课或学生论文参考来源 |

## 四、课堂问题模板

| 问题类型 | 示例 |
|---|---|
| 概念理解 | 作者如何定义“设计史”的研究对象？ |
| 历史分析 | 该文献如何解释现代设计与工业化的关系？ |
| 比较分析 | 这本文献与另一本文献对包豪斯的理解有何差异？ |
| 案例分析 | 这个案例可以说明现代设计教育的哪些特点？ |
| 批判讨论 | 作者的观点是否仍适用于当代数字设计？ |
| 方法训练 | 这篇文献使用了什么研究方法，适合如何转化为教材写作？ |

## 五、作业设计模板

| 作业类型 | 说明 |
|---|---|
| 阅读报告 | 摘要文献核心观点并说明可用于教材哪一章 |
| 文献比较 | 比较两篇文献对同一问题的不同解释 |
| 案例分析 | 选择一个设计作品或机构案例进行历史分析 |
| 概念卡片 | 整理一个核心概念的定义、来源与适用章节 |
| 小论文 | 围绕教材章节问题进行专题写作 |
| 资料卡建设 | 将文献转化为索引条目、观点条目和引文条目 |

## 六、解决的问题

这一层解决：这本文献怎样转化为教学内容，怎样帮助学生理解、讨论和写作。
