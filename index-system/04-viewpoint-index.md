# 04 核心观点索引

本索引用于提炼文献中的核心观点，服务教材写作中的定义、解释、批判、历史判断和方法论论证。

## 一、字段模板

| 观点编号 | 观点内容 | 所属文献编号 | 所属文献 | 页码 / 段落 | 观点类型 | 可用于教材位置 | 引用方式 | 关联观点 | 备注 |
|---|---|---|---|---|---|---|---|---|---|
| V001 |  | B001 |  |  | 定义型 | 第一章 第一节 | 直接引用 / 间接转述 | V002 |  |
| V002 |  | P001 |  |  | 批判型 | 方法论章节 | 间接转述 | V001 |  |

## 二、观点类型

| 类型 | 说明 | 教材用途 |
|---|---|---|
| 定义型 | 对概念、学科、对象作出定义 | 用于概念解释 |
| 解释型 | 解释历史现象或理论关系 | 用于正文展开 |
| 批判型 | 对既有观点、制度或叙述提出批评 | 用于学术辨析 |
| 历史判断型 | 对某一时期、事件、人物作出历史判断 | 用于章节结论 |
| 方法论型 | 提出研究方法或分析路径 | 用于导论与方法章节 |
| 案例归纳型 | 从案例中总结规律 | 用于案例教学 |
| 争议型 | 与其他观点存在分歧 | 用于课堂讨论 |

## 三、示例

| 观点编号 | 观点内容 | 来源 | 可用于章节 |
|---|---|---|---|
| V001 | 设计史不能只研究风格演变，还应研究设计与社会生活、技术系统和人工物环境之间的关系。 | 设计史理论文献 | 设计史方法论 |
| V002 | 现代设计的发展与工业化、消费文化、大众传播和社会组织方式密切相关。 | 现代设计史著作 | 现代设计章节 |

## 四、使用要求

1. 每条观点必须绑定到具体文献编号。
2. 用于正文直接引用的观点必须补充页码或段落位置。
3. 同一主题下相互支持或冲突的观点要建立关联。
4. 教材中应优先使用 A 类、B 类文献中的观点。

## 五、解决的问题

这一层解决：写教材时可以引用哪些观点来支撑论述，以及这些观点之间如何相互支持、修正或冲突。
