## 二、文献定位

该文是服务主导逻辑的经典文本，虽然属于营销学，但对服务设计、体验设计和设计管理非常重要，因为它把价值从“嵌入商品”转向“在使用和关系中共同创造”。

## 三、核心内容摘要

Vargo 与 Lusch 指出，传统营销建立在有形商品、离散交易和 4P 管理框架上；但 20 世纪末以来，服务营销、关系营销、质量管理、价值链、网络和资源理论正在推动新的主导逻辑。新逻辑把服务定义为通过知识和技能为他者创造利益的行动过程，认为货物只是服务传递机制，顾客不是被动目标，而是价值共创者。资源也从可占有的 operand resources 转向动态的 operant resources，如知识、能力和组织过程。

## 五、核心概念

| 概念 | 英文 | 教材解释方向 |
|---|---|---|
| 服务主导逻辑 | service-dominant logic | 把服务、过程、关系和价值共创作为经济交换核心的理论。 |
| 商品主导逻辑 | goods-dominant logic | 以有形产品、交易和价值嵌入为中心的传统营销逻辑。 |
| 价值共创 | value co-creation | 企业、用户和其他主体在使用与关系中共同形成价值。 |
| 操作性资源 | operant resources | 能作用于其他资源并产生效果的知识、技能和能力。 |

## 六、教材使用建议

可用于服务设计章节，说明服务设计的理论背景不是“无形产品设计”，而是价值、关系、过程和用户参与的重构。

## 七、教学转化

课堂问题：为什么用户不是价值的终端接收者，而是价值共创者？作业可把一个产品案例改写为服务主导逻辑下的价值网络。
